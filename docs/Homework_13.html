Q2: How does population affect gdp

Answer: There does not appear to be a strong relationship between gdp and population. There are some countries that have very high gdp per populations, especially in Asia. This was not a relationship I expected to be strong.

  mutate(
    log_pop = log(pop),
    log_gdpPercap = log(gdpPercap)

model2 <- lmer(
  log_gdpPercap ~ log_pop + year + continent + (1 | country),
  data = data
